Доступна версия 1Panel v2.1.11. Релиз сосредоточен на удобных улучшениях в Agents, File Browser, контейнерах и настройках панели, а также на исправлениях для обзора, сайтов, мультиузла и других модулей.

## Highlights

- **DeepSeek V4 в пулах моделей** — для учётных записей DeepSeek по умолчанию используются актуальные модели **V4**
- **Управление распаковкой в File Browser** — остановка текущих задач распаковки и более понятный интерфейс списка задач
- **VS Code по SSH** — уточнённые инструкции по подключению и расширенный скрипт настройки SSH
- **Загрузка логов контейнеров** — настроены таймауты; очистка процессов Docker после таймаута (см. Bug Fixes)
- **SSL панели для IPv6** — самоподписанный сертификат панели на серверах только с IPv6 или dual stack
- **Вёрстка терминала** — правки высоты внутренних элементов уменьшают лишнюю внешнюю прокрутку

## Enhancements

### Agents

- **Пулы моделей DeepSeek** — пулы учётных записей DeepSeek по умолчанию включают последние модели **V4** ([`#12622`](https://github.com/1Panel-dev/1Panel/pull/12622)).

### Containers

- **Таймауты загрузки логов** — время ожидания при загрузке логов контейнеров лучше соответствует сети и диску ([`#12632`](https://github.com/1Panel-dev/1Panel/pull/12632)).

### File Browser

- **Остановка распаковки** — задачи распаковки можно отменить во время выполнения; улучшен список задач для параллельной работы ([`#12582`](https://github.com/1Panel-dev/1Panel/pull/12582)).
- **VS Code и SSH** — уточнено руководство по подключению VS Code, расширено содержимое SSH-скрипта ([`#12613`](https://github.com/1Panel-dev/1Panel/pull/12613)).

### Terminal

- **Высота встроенного терминала** — правки вёрстки снижают появление лишней внешней полосы прокрутки ([`#12630`](https://github.com/1Panel-dev/1Panel/pull/12630)).

### Panel Settings

- **IPv6 и самоподписанный SSL** — HTTPS панели с самоподписанным сертификатом при доступе к серверу по IPv6 ([`#12652`](https://github.com/1Panel-dev/1Panel/pull/12652)).
