Доступна версия 1Panel v2.1.13. Релиз добавляет встроенный загрузчик моделей, улучшает File Browser и процесс обновления, а также содержит исправления для сред Runtime и настроек панели.

## Highlights

- **Загрузчик моделей** — загрузка и управление моделями в модуле Models без выхода из панели
- **File Browser** — понятнее ошибки истории файлов, быстрее расчёт размера каталогов с контролем параллелизма, удобнее множественный выбор
- **Безопасное обновление** — перед обновлением проверяется свободное место на диске
- **Runtime** — каталоги проектов не удаляются при неудачном создании среды; сопоставления хостов снова удаляются корректно
- **Настройки панели** — улучшена повторная привязка лицензии и восстановление снимков

## New Features

### Models

- **Загрузчик моделей** — загрузка моделей прямо в 1Panel ([`#12694`](https://github.com/1Panel-dev/1Panel/pull/12694)).

## Enhancements

### File Browser

- **Ошибки истории** — более предсказуемая обработка ошибок при загрузке содержимого истории ([`#12736`](https://github.com/1Panel-dev/1Panel/pull/12736)).
- **Размер каталогов** — оптимизирован расчёт с контролем параллелизма ([`#12737`](https://github.com/1Panel-dev/1Panel/pull/12737)).
- **Выбор файлов** — улучшены интерфейс управления файлами и операции выбора ([`#12740`](https://github.com/1Panel-dev/1Panel/pull/12740)).

### Panel Settings

- **Повторная привязка лицензии** — улучшена логика в ряде сценариев ([`#12778`](https://github.com/1Panel-dev/1Panel/pull/12778)).

### System

- **Проверка диска при обновлении** — перед продолжением проверяется свободное место ([`#12708`](https://github.com/1Panel-dev/1Panel/pull/12708)).

## Bug Fixes

| Область | Исправление |
|---|---|
| **Runtime** | Каталоги проектов больше не удаляются при неудачном создании среды ([`#12756`](https://github.com/1Panel-dev/1Panel/pull/12756)). |
| **Runtime** | Сопоставления хостов среды снова можно удалить ([`#12779`](https://github.com/1Panel-dev/1Panel/pull/12779)). |
| **File Browser** | Загрузка несуществующего каталога не вызывает неожиданных ошибок ([`#12744`](https://github.com/1Panel-dev/1Panel/pull/12744)). |
| **Panel Settings** | Восстановление снимков исправлено в сценариях, где ранее завершалось сбоем ([`#12777`](https://github.com/1Panel-dev/1Panel/pull/12777)). |

Нажмите кнопку `Update` в правом нижнем углу панели.
