Доступна версия 1Panel v2.1.12. Релиз улучшает экран Overview и содержит точечные исправления для File Browser, терминалов на дочерних узлах и обновления панели на OpenWRT.

## Highlights

- **Диаграммы мониторинга** — графики на главной странице читаются нагляднее
- **Заметки и карусель** — карусель на главной корректно останавливается при редактировании заметок
- **File Browser** — более надежная распаковка ZIP, сохранение прав родительского каталога после распаковки и загрузки, исправление при перемещении очень больших бинарных файлов
- **Терминалы дочерних узлов** — подключения к хосту по умолчанию снова ведут себя ожидаемо
- **OpenWRT** — обновление панели снова завершается успешно на системах на базе OpenWRT

## Enhancements

### Overview

- **Отображение диаграмм мониторинга** — улучшены компоновка и читаемость диаграмм на главной ([`#12673`](https://github.com/1Panel-dev/1Panel/pull/12673)).

## Bug Fixes

| Область | Исправление |
|---|---|
| **Overview** | Карусель на главной приостанавливается при редактировании заметок ([`#12680`](https://github.com/1Panel-dev/1Panel/pull/12680)). |
| **File Browser** | Распаковка отдельных ZIP-архивов больше не завершается сбоями ([`#12684`](https://github.com/1Panel-dev/1Panel/pull/12684)). |
| **File Browser** | Права родительского каталога не меняются после распаковки ([`#12684`](https://github.com/1Panel-dev/1Panel/pull/12684)). |
| **File Browser** | Перемещение очень больших бинарных файлов не приводит к недоступности сервера ([`#12683`](https://github.com/1Panel-dev/1Panel/pull/12683)). |
| **File Browser** | Права родительского каталога сохраняются после загрузки файлов ([`#12672`](https://github.com/1Panel-dev/1Panel/pull/12672)). |
| **Terminal** | Для терминалов на дочерних узлах восстановлены корректные подключения к хосту по умолчанию ([`#12676`](https://github.com/1Panel-dev/1Panel/pull/12676)). |
| **System** | Обновления 1Panel снова работают на OpenWRT ([`#12670`](https://github.com/1Panel-dev/1Panel/pull/12670)). |

## How to Upgrade

Нажмите кнопку `Update` в правом нижнем углу панели.

Впервые знакомитесь с 1Panel? Сайт [1panel.pro](https://1panel.pro/).
