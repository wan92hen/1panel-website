1Panel v2.1.6 was officially released on 2026-03-26. This release focuses on better AI workflows, stronger agent capabilities, and smoother daily operations.

## Highlights

- New AI Terminal that converts natural language into executable shell commands
- Agent skill management and skill store support
- New WeChat channel support for agents
- DingTalk channel support for agents

## New Features

- Terminal: AI Terminal added
- Agent: skill management added
- Agent: WeChat and DingTalk channels added
- Agent: Xiaomi model account support
- Agent: custom npm registry support
- Agent: model pool configuration support
- Agent: overview information support
- Agent: main config editing support

## Optimizations

This version also improves plugin installation logic, OpenClaw terminal experience, model account validation, reverse proxy SSL skip checks, container file navigation, file upload/download behavior, multi-node UI, and sidebar consistency.

## Bug Fixes

The team fixed known issues in App Store, websites, certificates, containers, files, terminal, toolbox, audit logs, panel settings, and system UI behavior.
