## What Is 1Panel?

1Panel is a modern, open-source VPS control panel built for Linux servers. It provides a clean web interface for file management, database administration, Docker operations, SSL certificates, and one-click app deployment.

It is free under an open-source license, with an optional Pro plan for advanced capabilities.

1Panel aligns with AI-era deployment patterns. Teams can manage AI-related services and model operations with significantly less operational friction than legacy panel workflows.
