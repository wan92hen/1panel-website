Since OpenClaw launched in the 1Panel App Store, the project has received strong community adoption. This update note introduces OpenClaw v2026.2.1 and explains the one-click upgrade flow in 1Panel.

## Why Upgrade Through 1Panel

Compared with manual image replacement or command-line updates, upgrading through 1Panel provides:

- No need to log in and run shell commands on the server.
- No manual image pulling and file replacement process.
- Fully visual operation in the App Store interface.
- Built-in backup and comparison flow for safer upgrades.
- Fast rollback support if an upgrade issue occurs.

## Upgrade Steps

1. Open **App Store** in 1Panel.
2. Check **Upgradable** apps for OpenClaw.
3. If no prompt appears, click **Update Remote Apps** and re-check.
4. Click **Upgrade** on OpenClaw.
5. Choose target version, optional backup settings, and apply.
