In early March 2026, 1Panel released v2.1.3 and v2.1.4. This update round focused on Agent, Local Models, File Management, Database, Container, and System capabilities.

## New Features

- Agent: add QQ channel support.
- Agent: add OpenRouter model account support.
- Local Models: add vLLM management.
- File Management: support showing/hiding columns in file lists.

## Improvements

- Agent: improved save behavior for custom model name configuration.
- Certificate: selector now shows issuer and expiration date.
- Database: optimized PostgreSQL delete logic.
- Container: optimized Docker restart prompt information.
- Panel Settings: improved panel SSL page prompts.

## Bug Fixes

- App Store: fixed sorting issues for newly installed apps.
- Website: fixed reverse-proxy creation problems in specific scenarios.
- Container: fixed compose creation failures in some cases.
- Container: fixed env file being cleared when compose path is manually pasted.
- Logs: fixed abnormal log display for some child nodes.
- Panel Settings: fixed email receive failure when sender name uses Chinese characters.
- System: fixed known international version issues and menu flicker during language switching.
- System: fixed missing translations in some languages.
