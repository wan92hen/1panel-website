1Panel v2.1.0 於 2026-02-12 正式發佈，本次更新聚焦於 AI/Agent 工作流程、面板設定與系統安全。

## 版本亮點

- OpenClaw Agent 管理正式整合進 1Panel。
- 面板設定新增企業告警通道（企業微信、釘釘、飛書；X-Pack）。
- 支援 Passkey 登入，降低對密碼登入的依賴。
- 概覽模組新增 Memo 功能。

## 主要新增

- 概覽：新增 Memo。
- AI：新增 OpenClaw Agent 管理。
- 證書：支援自訂 ACME 帳號驗證。
- 容器：支援批量匯入映像。
- 檔案：支援檔案管理備註。
- WAF（X-Pack）：支援匯出攔截紀錄。
- 面板設定（X-Pack）：支援企業微信、釘釘、飛書通知。
- 系統：支援 Passkey 登入。

## 其他優化與修復

本版同時改善了 App Store 同步與快取、網站重定向與清單顯示、容器建立體驗、檔案下載記憶體佔用，以及多項已知問題。
