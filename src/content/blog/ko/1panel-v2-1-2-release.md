2026년 3월 4일 1Panel v2.1.2가 정식 출시되었습니다. 앱 스토어, 에이전트, 웹사이트, 컨테이너, 시스템 전반이 업데이트되었습니다.

## 주요 기능

- 설치된 앱 정렬 지원
- CoPaw 타입 에이전트 생성 지원
- Telegram/Discord 채널 추가
- 에이전트 브라우저/시간대 설정 지원
- 모델 계정 연동 확장
- 터미널 글꼴 및 전경/배경 색상 커스터마이징
- 패널 실행 리전 전환 지원
