1Panel v2.1.6 se lanzó oficialmente el 26/03/2026. Esta versión mejora la experiencia de IA y amplía las capacidades de agentes.

## Novedades destacadas

- Nuevo terminal de IA que traduce lenguaje natural a comandos shell
- Gestión de skills y tienda de skills para agentes
- Nuevo soporte de canal de WeChat para agentes
- Soporte de canal de DingTalk para agentes

## Nuevas funciones

- Terminal: función de terminal de IA
- Agentes: gestión de skills
- Agentes: canales WeChat y DingTalk
- Agentes: soporte para cuenta de modelo Xiaomi
- Agentes: soporte para repositorio npm personalizado

## Optimizaciones y correcciones

Se optimizaron módulos de sitio web, contenedores, archivos, terminal y sistema; además se corrigieron múltiples errores en App Store, certificados, registros y configuración del panel.
