1Panel v2.1.0 は 2026年2月12日に正式リリースされ、AI 管理、通知、セキュリティ、日常運用の効率化に重点を置いた更新が行われました。

## 主要アップデート

- OpenClaw エージェント管理を追加。
- WeCom / DingTalk / Feishu 通知（X-Pack）に対応。
- Passkey ログインを追加。
- 概要ページにメモ機能を追加。

## そのほか

- 証明書、コンテナ、ファイル管理、WAF などを強化。
- Web サイト/アプリストア/システム周辺の改善と不具合修正を多数実施。
